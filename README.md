# Project Start

1. Install dependencies

- Back-End
````````````
composer install
````````````
- Front-End
````````````
npm install or yarn install
````````````

2. Run Laravel Back-End

````````````
php artisan serve
````````````

3. Run Vue3 Front-End

````````````
npm run dev
````````````
